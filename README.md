# アルミオシオンの医術師 - 天退星刀模組 (含專屬BGM)

本模組將武器「天退星刀」、全套專屬技能、狀態堆疊機制（燒傷、震顫與引爆）以及戰鬥 BGM 自動切換系統，不需修改遊戲原始 JSON 資料庫。

---

## 模組介紹

本模組圍繞「天退星刀」建立了一套獨特的戰鬥資源機制：
* **彈藥堆疊與轉化**：裝備武器後，戰鬥開始時自帶「虎標彈」，耗盡後自動裝填為「猛虎標彈」並獲得「天退星」狀態。累積消耗達標後，進一步進化為「心-天退星」。
* **過熱反噬機制**：彈藥全數耗盡後進入「過熱」階段，隨著消耗總量提升，將解鎖傷害減免與極高幅度的攻擊力加成。
* **震顫引爆系統**：技能可對敵方施加「燒傷」與「震顫」層數。特定高階技能會將震顫轉化為「震顫-灼熱」，並引爆層數造成固定傷害。
* **自動連動文本**：所有技能與狀態的說明文字皆與腳本內部的數值設定即時連動，修改設定後無須手動更改文本。

## 安裝辦法

### 1. 下載模組檔案
請至本專案右側的 **Releases** 區塊，下載最新版本的 `.zip` 壓縮包。

### 2. 部署遊戲目錄
解壓縮後，請將檔案依據以下路徑放入遊戲資料夾中：
* 將 `Mod_TianTuiXing.js` 放入：`遊戲資料夾/js/plugins/`
* 將 `Mod_Lei_Heng_Theme.ogg` 放入：`遊戲資料夾/audio/bgm/`
* 打開 `遊戲資料夾/js/plugins.js` ，在其中的 `var $plugins =[ ` 後面加上一行  `{"name":"Mod_TianTuiXing","status":true,"description":"天退星刀","parameters":{}},`

> ⚠️ **注意事項**
> * 進入新遊戲或讀取舊存檔時，系統若偵測到隊伍與背包中無此武器，將會**自動發放一把「天退星刀」**至玩家背包中。
> * 專屬戰鬥 BGM 具備偵測功能，在戰鬥中裝備該武器會立刻切換音樂，解除裝備則會還原為系統預設戰鬥音樂。

## 玩家自訂指南

若需調整武器傷害、技能冷卻、彈藥數量或 BGM 檔名，請直接使用文字編輯器（如 Notepad++、VS Code）打開 `Mod_TianTuiXing.js`。

請修改腳本上半部的 **【模組核心設定區 (TIAN_MOD)】**：

```javascript
// 你可以在此處自由修改數值，遊戲內說明文字會自動同步轉化
const TIAN_MOD = {
    ids: {
        weapon: 530,       // 武器 ID
        // ... 其他技能與狀態 ID
    },
    bgm: {
        name: "Mod_Lei_Heng_Theme", // BGM 檔名 (不含副檔名)
        volume: 90,                 // 預設音量
        pitch: 100                  // 音調
    },
    params: {
        weaponStats: [0, 0, 200, 0, 0, 0, 100, 0], // 武器屬性加成
        // ... 調整傷害公式與冷卻回合
    }
};
```

詳細請參考js文件內說明。
