# アルミオシオン的医术师 - 天退星刀模组 (含专属BGM)

本模组整合了武器“天退星刀”、全套专属技能、状态叠加机制（烧伤、震颤与引爆）以及战斗 BGM 自动切换系统，无需修改游戏原始 JSON 数据库。

---

## 模组介绍

本模组围绕“天退星刀”建立了一套战斗资源机制：
* **弹药叠加与转化**：装备武器后，战斗开始时自带“虎标弹”，耗尽后自动装填为“猛虎标弹”并获得“天退星”状态。累计消耗达标后，进一步进化为“心-天退星”。
* **过热反噬机制**：弹药全数耗尽后进入“过热”阶段，随着消耗总量提升，将解锁伤害减免与极高幅度的攻击力加成。
* **震颤引爆系统**：技能可对敌方施加“烧伤”与“震颤”层数。特定高阶技能会将震颤转化为“震颤-灼热”，并引爆层数造成固定伤害。
* **自动联动文本**：所有技能与状态的说明文字均与脚本内部的数值设置即时联动，修改设置后无需手动更改文本。

## 安装指南

请选择以下其中一种方法将模组安装至您的游戏中：

### 方法一：通过 Release 压缩包安装（推荐，含 BGM 音频文件）
1. **下载模组文件**：请至本项目右侧的 Releases 区域，下载最新版本的 `.zip` 压缩包。
2. **部署游戏目录**：解压后，将文件依据以下路径放入游戏文件夹中：
   * 将 `Mod_TianTuiXing.js` 放入：`游戏文件夹/js/plugins/`
   * 将 `Mod_Lei_Heng_Theme.ogg` 放入：`游戏文件夹/audio/bgm/`
3. **手动注册插件**：打开 `游戏文件夹/js/plugins.js`，在其中的 `var $plugins = [` 后面加上一行：
   `{"name":"Mod_TianTuiXing","status":true,"description":"天退星刀","parameters":{}},`

### 方法二：手动创建文件与复制源码（适合无压缩包、纯脚本安装）
1. **新建脚本文件**：直接进入你的 `游戏文件夹/js/plugins/` 目录，在此处新建一个文本文件，并重命名为 `Mod_TianTuiXing.js`（请确保文件后缀名为 `.js` 而非 `.txt`）。
2. **粘贴模组源码**：使用文本编辑器（如记事本、VS Code）打开该文件，将本项目中的 `Mod_TianTuiXing.js` 代码完整复制并粘贴至其中后保存。
3. **手动注册插件**：打开 `游戏文件夹/js/plugins.js`，在其中的 `var $plugins = [` 后面加上一行：
   `{"name":"Mod_TianTuiXing","status":true,"description":"天退星刀","parameters":{}},`

⚠️ **注意事项**

* **音乐配置**：若使用方法二安装，请务必自行准备对应的 BGM 音乐文件（默认银/文件名为 `Mod_Lei_Heng_Theme.ogg`）并放入 `游戏文件夹/audio/bgm/` 中，否则触发战斗时可能会因找不到音乐而报错。
* **自动发放**：进入新游戏或读取旧存档时，系统若检测到队伍与背包中无此武器，将会**自动发放一把“天退星刀”**至玩家背包中。
* **动态 BGM**：专属战斗 BGM 具备检测功能，在战斗中装备该武器会立刻切换音乐，卸载装备则会还原为系统默认战斗音乐。

## 玩家自定义指南

若需调整武器伤害、技能冷却、弹药数量或 BGM 文件名，请直接使用文本编辑器（如 Notepad++、VS Code）打开 `Mod_TianTuiXing.js`。

请修改脚本上半部分的 **【模组核心设置区 (TIAN_MOD)】**：

```javascript
// 你可以在此处自由修改数值，游戏内说明文字会自动同步转化
const TIAN_MOD = {
    ids: {
        weapon: 530,       // 武器 ID
        // ... 其他技能与状态 ID
    },
    bgm: {
        name: "Mod_Lei_Heng_Theme", // BGM 文件名 (不含后缀名)
        volume: 90,                  // 默认音量
        pitch: 100                   // 音调
    },
    params: {
        weaponStats: [0, 0, 200, 0, 0, 0, 100, 0], // 武器属性加成
        // ... 调整伤害公式与冷却回合
    }
};
```
详情请参考 js 文件内说明。
